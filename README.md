# Dotfiles
This is my dotfiles.

## Usage
```bash
git clone --recursive https://github.com/nahcnuj/dotfiles.git
cd dotfiles
make install

# for Windows Subsystem for Linux
sudo ln -sfnv $HOME/etc/wsl.conf /etc/wsl.conf
```
